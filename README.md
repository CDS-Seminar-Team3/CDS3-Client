<summary> 1️⃣ Commit 컨벤션  </summary>

* 기본적인것 (feat, fix, chore, ..) 위주로만 지키기!

| 제목     | 내용                                            |
| -------- | -------------------------------------------    |
| init     | 브랜치 첫 커밋                                  |
| feat     | 새로운 기능을 추가할 경우                        |
| fix      | 버그를 고친 경우                                |
| chore    | 그 외 자잘한 수정에 대한 커밋(주석추가 문서 수정) |
| docs     | 문서 수정에 대한 커밋                           |
| style    | style: 코드 스타일 혹은 포맷 등에 관한 커밋      |
| refactor | 코드 리팩토링에 대한 커밋                        |
</details>
